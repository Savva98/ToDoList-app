Features

Everything is simple. Just go ahead and make your own list.

Add the task to your list

When you fill up form and press 'Upload' button, all the tasks you have wrote, will be moved to the task list.

Mark the task as a done

When you press the complete button the task will be removed from the list, but you can still see how much task you have already completed, if you hover the mouse on the "It is done button".

Delete the task, whenever you like

When you click delete button, task will be removed from the task list. You can also remove tasks when you filling the form if you misspell something.
